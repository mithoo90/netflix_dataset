# netflix_dataset
This project focuses on analyzing the Netflix dataset using Structured Query Language (SQL) to uncover insights about movies, TV shows, genres, release patterns, and content distribution across countries and time. The goal is to apply advanced SQL techniques to perform real-world data analysis
Key Objectives

Perform data exploration and cleaning using SQL.

Extract meaningful insights such as:

Most common genres and directors.

Country-wise content distribution.

Trends in movie vs. TV show releases.

Content growth over time.

Use SQL aggregate functions, subqueries, window functions, and CTEs to build complex analytical queries.

Simulate a real-world data analysis workflow entirely in SQL.

Tools & Technologies

SQL (Structured Query Language)

 PostgreSQL 

Dataset: Netflix Titles Dataset (from Kaggle or other open-source sources)

 Skills Demonstrated



Exploratory data analysis using SQL

Query optimization and logical reasoning

Data storytelling through SQL insights
